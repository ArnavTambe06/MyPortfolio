# Arnav Portfolio

A personal portfolio built with Next.js, React, and Tailwind CSS. It presents selected projects, experience, resume details, and contact links in a clean one-page layout.

## Features

- Hero section with intro and navigation
- About, projects, experience, resume, and contact sections
- Responsive layout for desktop and mobile
- Smooth motion and modern portfolio styling
- External links to GitHub, LinkedIn, email, and live project demos

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - build the app for production
- `npm run start` - start the production server
- `npm run lint` - run ESLint

## Project Structure

- `app/page.tsx` - main landing page
- `app/components/` - reusable portfolio sections
- `app/globals.css` - global styles and theme tokens

## Deploy

This project is ready to deploy on Vercel or any platform that supports Next.js.
